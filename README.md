# Accounting App

一個簡單的本機記帳網頁工具，使用 HTML、CSS、JavaScript 製作，不需要後端或資料庫。

## 功能

- 新增收入與支出
- 分類管理
- 付款方式紀錄
- 月度收入、支出、結餘統計
- 支出分類圖表
- 搜尋與篩選交易紀錄
- 刪除交易紀錄
- 匯出 JSON / CSV
- 匯入 JSON

## 使用方式

直接打開 `index.html` 即可使用。

資料會儲存在瀏覽器的 `localStorage`，不會上傳到伺服器。
